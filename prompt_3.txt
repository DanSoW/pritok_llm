Напиши программу, которая будет обрабатывать изображение, представленное в виде массива 255x255 целого типа. Значения для данного массива (изображения) должны быть сгенерированы рандомно от 0 до 255 включительно. Добавь проверку массива значений изображение и если сумма значений элементов главной диагонали равна сумме значений элементов побочной диагонали, то изображение не прошло проверку, в противном случае изображение прошло проверку. Используй язык программирования C++
